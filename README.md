## Run using `docker compose`

### Prerequisites

1. Docker
2. Docker Compose v2

From mypics folder run the following command:
```bash
docker compose up
```

You can check if the containers are running using:
```bash
docker ps
```

The frontend is now running on `http://localhost:3000`
